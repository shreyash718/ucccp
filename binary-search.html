Concept

Divide & Conquer on Sorted Data

Binary search finds a target in a sorted array by repeatedly halving the search space. Check the middle element — if it's too big, go left; too small, go right.

Each step eliminates half the remaining elements → O(log n) total steps.

Linear Search: O(n)
Binary Search: O(log n)
Requires: sorted array
int binarySearch(vector<int>& arr, int target) {
    int lo = 0, hi = arr.size() - 1;
    while(lo <= hi) {
        int mid = lo + (hi - lo) / 2; // avoids overflow
        if(arr[mid] == target) return mid;
        else if(arr[mid] < target) lo = mid + 1;
        else hi = mid - 1;
    }
    return -1; // not found
}

Pro tip: C++ STL has lower_bound and upper_bound — use them in contests!
