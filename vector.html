Concept

What is a Vector?

A vector (or dynamic array) is an array that can grow or shrink in size automatically. Unlike a fixed-size array, you don't need to know the size in advance.

Internally, when the array is full it allocates a new block (usually 2× the current capacity), copies existing elements, and frees the old block. This "doubling" strategy gives amortized O(1) push_back.

  • push_back(x) — append element → O(1) amortized
  • pop_back() — remove last element → O(1)
  • v[i] — random access → O(1)
  • size() — current count → O(1)
  • clear() — remove all elements → O(n)
Access: O(1)
Push: O(1) amortized
Pop: O(1)
Space: O(n)
#include <bits/stdc++.h>
using namespace std;
int main() {
    vector<int> v;           // empty vector
    v.push_back(10);        // [10]
    v.push_back(20);        // [10, 20]
    v.push_back(30);        // [10, 20, 30]
    v.pop_back();           // [10, 20]
    cout << v[0];           // prints 10
    cout << v.size();       // prints 2
    // iterate
    for(int x : v) cout << x << " ";
}
Practice Problem

Find Second Largest Element

Given an array of integers, find the second largest element. Use a vector to store input, then iterate to find the answer.

Input: [3, 1, 4, 1, 5, 9, 2, 6]  →  Output: 6

#include <bits/stdc++.h>
using namespace std;
int main() {
    vector<int> v = {3,1,4,1,5,9,2,6};
    int first = INT_MIN, second = INT_MIN;
    for(int x : v) {
        if(x > first) {
            second = first;
            first = x;
        } else if(x > second && x != first) {
            second = x;
        }
    }
    cout << "Second Largest: " << second; // 6
}
